Node  parent 
)

check if one node is child of another node

We define the notion of child node as the following. For all e in E[G] and for all {v,w} in V[e] if e is an incoming edge of v and outgoing edge of w than v is child of w. As you can see from the definition provided in is_parent_of() as well, if v is child of w, then w is parent of v.
