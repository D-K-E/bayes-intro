557 The idea here is the following: For the last variable eliminated, Z, the factor for the value x contains the probability of the most likely assignment that contains Z=x. For example: let's say g* = argmax(psi(G))

  1. l* = argmax(psi[g*](L))
  2. d* = argmax(psi[l*](D))
